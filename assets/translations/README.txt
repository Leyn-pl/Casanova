How to translate:
Edit translation.csv in your favourite csv editor
Save as UTF-8, with comma separators

Make a Pull Request with the changed translation.csv file.
The other files do not have to be edited as they are generated automatically, based off
of translation.csv



Adding new bundles:

1. Find your locale symbol here: https://docs.godotengine.org/en/3.1/tutorials/i18n/locales.html
2. Add a new row, with the locale symbol as the key
3. Add your translations
